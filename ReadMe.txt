What is this?
This is a remake of Dexter's Laboratory: Robot Rampage made by SeriousCsaba.

What is Dexter's Laboratory: Robot Rampage?
Sky Gamestar was a UK television game provider. It was a service in which Sky TV customers could play interactive games, using a TV remote as a controller. It was shut down on January 31st, 2015. Many games on the platform are now lost media, including Dexter's Laboratory: Robot Rampage.
